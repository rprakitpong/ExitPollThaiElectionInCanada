# VancouverExitPoll2569

Vancouver exit poll of Thai general election 2569/2026

Exit Poll ผลแบบคูหา ที่สถานกงสุลใหญ่ฯ ในวันที่ 24-25 ม.ค. 2569 เวลา 10.00-16.00 น.

## สาระสังเขป

การเลือกตั้งล่วงหน้า ณ สถานกงสุลใหญ่ ณ นครแวนคูเวอร์ ประเทศแคนาดา ในวันที่ 24-25 มกราคม 2569 มีผู้มาใช้สิทธิ์เลือกตั้ง มีผู้ตอบแบบสอบถามผลแบบคูหา (Exit Poll) จำนวน 529 คน โดยข้อมูลที่ได้จากการสำรวจนี้ สามารถจำแนกออกเป็น 2 ส่วนหลัก ได้แก่ การเลือกตั้งแบบบัญชีรายชื่อพรรคการเมือง และการลงประชามติแก้ไขรัฐธรรมนูญแห่งราชอาณาจักรไทย พ.ศ. 2560

ผลสรุปจากคำถาม "ในการเลือกตั้งรอบนี้ คุณได้กาบัญชีรายชื่อให้พรรคใด" พบว่า พรรคประชาชนได้รับคะแนนนิยมสูงสุดที่ 456 คะแนน รองลงมาคือ พรรคภูมิใจไทย 17 คะแนน พรรคประชาธิปัตย์ 12 คะแนน พรรคเพื่อไทย 11 คะแนน พรรคไทยสร้างไทย 4 คะแนน พรรครวมไทยสร้างชาติ 5 คะแนน พรรคอื่นๆ 8 คะแนน และไม่ประสงค์ลงคะแนน 16 คะแนน

สำหรับผลการลงประชามติแก้ไขรัฐธรรมนูญแห่งราชอาณาจักรไทย พ.ศ. 2560 พบว่า ผู้ตอบแบบสอบถามส่วนใหญ่ 477 คน เห็นด้วยกับการแก้ไขรัฐธรรมนูญ ส่วน 28 คน ไม่เห็นด้วยกับการแก้ไขรัฐธรรมนูญ และ 19 คน ไม่ประสงค์ลงคะแนน หรือไม่มีสิทธิ์ลงคะแนนในครั้งนี้
<img width="1500" height="900" alt="exitpoll_YVR_Kawin_2026-01-26_referendum" src="https://github.com/user-attachments/assets/a645075d-c8bd-4ca9-8a85-bd15b662fac7" />

นอกจากนี้ ยังได้วิเคราะห์ความสัมพันธ์ระหว่างพรรคการเมืองที่ผู้ตอบแบบสอบถามเลือก กับความคิดเห็นต่อการแก้ไขรัฐธรรมนูญแห่งราชอาณาจักรไทย พ.ศ. 2560 พบว่า ผู้ที่เลือกพรรคประชาชน มีแนวโน้มที่จะเห็นด้วยกับการแก้ไขรัฐธรรมนูญมากที่สุด ในขณะที่ผู้ที่เลือกพรรคภูมิใจไทย มีแนวโน้มที่จะไม่เห็นด้วยกับการแก้ไขรัฐธรรมนูญมากกว่า และเป็นที่น่าสนใจว่า คนที่เลือกพรรคเพื่อไทย ทุกคนเห็นด้วยกับการแก้ไขรัฐธรรมนูญ

จำนวนคนตอบคิดเป็นสัดส่วนยังไม่สามารถวัดได้ เนื่องจากต้องรอให้ทางกงสุลใหญ่ฯ สรุปยอดผู้มาใช้สิทธิ์เลือกตั้งทั้งหมดก่อน อย่างไรก็ตาม ผลการสำรวจนี้สะท้อนให้เห็นถึงแนวโน้มและทิศทางของผู้มีสิทธิ์เลือกตั้งในแวนคูเวอร์ได้เป็นอย่างดี

## วิธีการเก็บข้อมูล

การเก็บข้อมูลทำโดยการแจกแบบสอบถามให้กับผู้มาใช้สิทธิ์เลือกตั้งล่วงหน้าที่สถานกงสุลใหญ่ ณ นครแวนคูเวอร์ ในวันที่ 24-25 มกราคม 2569 เวลา 10.00-16.00 น. โดยมีอาสาสมัครจากชุมชนไทยในแวนคูเวอร์เป็นผู้แจกแบบสอบถามและช่วยตอบคำถามเกี่ยวกับแบบสอบถามแก่ผู้มาใช้สิทธิ์เลือกตั้ง แบบสอบถามประกอบด้วยคำถามหลัก 2 ข้อ ได้แก่

1. ในการเลือกตั้งรอบนี้ คุณได้กาบัญชีรายชื่อให้พรรคใด
2. คุณเห็นด้วยหรือไม่กับการแก้ไขรัฐธรรมนูญแห่งราชอาณาจักรไทย พ.ศ. 2560

วิธีการเก็บข้อมูลในการเลือกตั้งปี 2569 นี้ แตกต่างจากการเก็บข้อมูลในการเลือกตั้งปี 2566 ที่ผ่านมา โดยในปี 2566 เป็นการตั้งคอมพิวเตอร์ในอาคารของ UBC ที่ Robson Square โดยใช้วิธี [keylogger code :: https://github.com/rprakitpong/ExitPollThaiElectionInCanada/tree/main/2566_vancouver](https://github.com/rprakitpong/ExitPollThaiElectionInCanada/tree/main/2566_vancouver) ในการเก็บข้อมูล

แต่ในปี 2569 นี้ ได้เปลี่ยนมา [PWA Web Application :: https://github.com/gkawin/exitpoll](https://github.com/gkawin/exitpoll) บน Tablet แทน อันเนื่องมาจากข้อจำกัดด้านสถานที่ และการเข้าถึงแหล่งไฟฟ้า ทำให้ไม่สามารถตั้งคอมพิวเตอร์ได้เหมือนเดิม อีกทั้งการใช้ Tablet ยังช่วยให้การเก็บข้อมูลเป็นไปอย่างรวดเร็วและมีประสิทธิภาพมากขึ้นอีกด้วย

ตัวเลือกพรรคการเมืองในแบบสอบถามปี 2569 นี้ ได้มีการปรับปรุงให้สอดคล้องกับสถานการณ์ทางการเมืองปัจจุบัน โดยได้ยึดรายชื่อพรรคการเมืองที่มาดีเบตในรายการ [TheStandardDebate ECONOMIC BATTLE ศึกทางออกเศรษฐกิจไทย | POLICY BATTLE ศึกดวลนโยบาย #1](https://www.youtube.com/watch?v=8XoCdn_O_cs&list=PLZ0vRaBzdrQ2bymt8pMqLDjQ_jTSZ-shI&index=1) หลังจากเพิ่มตัวเลือกแล้ว รู้สึกว่าหน้่าจอแบบสอบถามยังมีพื้นที่ว่างอยู่ จึงได้เพิ่มตัวเลือก "พรรครวมไทยสร้างชาติ", "พรรคอื่นๆ" และ "ไม่ประสงค์ลงคะแนน" เข้าไปด้วย เพื่อให้ครอบคลุมความหลากหลายของผู้ตอบแบบสอบถามมากยิ่งขึ้น

## ข้อสังเกตระหว่างเก็บข้อมูล

- __เริ่มเก็บข้อมูลหลังเปิดคูหาเล็กน้อย__ ในวันเสาร์–อาทิตย์เริ่มทำ Exit Poll หลังเปิดคูหาไปแล้วประมาณ 5 นาที ทำให้ยังไม่สามารถเก็บข้อมูลจากผู้มาใช้สิทธิ์กลุ่มแรกได้ครบถ้วน

- __ผู้มาใช้สิทธิ์บางส่วนไม่สะดวกเข้าร่วม__ พบว่ามีผู้มาใช้สิทธิ์บางรายปฏิเสธการตอบแบบสอบถาม โดยให้เหตุผลเกี่ยวกับความไม่ไว้วางใจต่อการจัดทำแบบสำรวจ

- __มีความเข้าใจคลาดเคลื่อนเกี่ยวกับที่มาของโพล__ ผู้ตอบบางท่านเข้าใจว่าโพลอาจมีความเกี่ยวข้องกับพรรคการเมือง เนื่องจากคำอธิบายว่าเป็น “โพลภาคประชาชนในแคนาดา” ซึ่งอาจทำให้ตีความไปเป็นชื่อพรรคการเมืองได้

- __บริบทการมาเป็นกลุ่มอาจกระทบการตัดสินใจตอบ__ กรณีผู้มาใช้สิทธิ์มาเป็นคณะ/กลุ่มเพื่อน บางคนเลือกไม่ตอบ เพราะกังวลเรื่องการเปิดเผยจุดยืนทางการเมืองต่อคนในกลุ่ม

- __การยืนยันการส่งคำตอบยังไม่ชัดเจน__ บางท่านไม่แน่ใจว่าทำแบบสอบถามเสร็จสมบูรณ์แล้ว เนื่องจากไม่มีสัญญาณ/ตัวบ่งชี้สถานะที่ชัดเจน ทำให้ทีมงานต้องช่วยตรวจสอบเพิ่มเติม

- __ประสบการณ์จากการเก็บข้อมูลครั้งก่อนมีผลต่อความร่วมมือ__ มีผู้ตอบบางรายเคยไม่เข้าร่วมในปี 2566 จากความกังวลเรื่องความเป็นกลางของผู้จัดทำ แต่เมื่อได้รับคำอธิบายเพิ่มเติมเกี่ยวกับวัตถุประสงค์และความเป็นกลาง จึงตัดสินใจเข้าร่วมในปีนี้

- __ความเป็นส่วนตัวยังเป็นประเด็นที่พบในภาคสนาม__ การทำแบบสอบถามในพื้นที่หน้างานอาจทำให้ผู้ตอบบางท่านรู้สึกไม่เป็นส่วนตัวหรือไม่สะดวกใจ เนื่องจากมีผู้อื่นอยู่ใกล้บริเวณหน้าจอขณะตอบแบบสอบถาม

## การวิเคราะห์ข้อมูล

ข้อมูลที่ได้จากการเก็บแบบสอบถาม ถูกส่งออกมาในรูปแบบ CSV จากนั้นจึงนำมาวิเคราะห์ด้วยภาษา Python โดยใช้ไลบรารี Pandas สำหรับการจัดการข้อมูล และ Matplotlib กับ Seaborn สำหรับการสร้างกราฟและแผนภูมิในการนำเสนอผลลัพธ์
