## การวิเคราะข้อมูล

การวิเคราะห์ข้อมูลได้ใช้ ChatGPT เขียนโค้ด Python มาเพื่อกราฟข้อมูล โดยชื่อตัวเลือกภาษาอังกฤษในกราฟได้ตรงกับชื่อไทยดังต่อไปนี้

1. ในการเลือกตั้งรอบนี้ คุณได้กาบัญชีรายชื่อให้พรรคใด

- PEOPLE: พรรคประชาชน
- BHUMJAITHAI: พรรคภูมิใจไทย
- DEMOCRAT: พรรคประชาธิปัตย์
- PHEU_THAI: พรรคเพื่อไทย
- THAI_SANG_THAI: พรรคไทยสร้างไทย
- UTN: พรรครวมไทยสร้างชาติ
- OTHER: พรรคอื่นๆ
- NO_VOTE: ไม่สะดวกตอบ/งดออกเสียง/ไม่ได้ลงทะเบียน/ไม่มีสิทธิ์

2. คุณเห็นด้วยหรือไม่กับการแก้ไขรัฐธรรมนูญแห่งราชอาณาจักรไทย พ.ศ. 2560

- AGREE: เห็นชอบ
- DISAGREE: ไม่เห็นชอบ
- NO_RIGHT: ไม่มีสิทธิ์
- N/A: ไม่สะดวกตอบ

กราฟของผลจากคำถามทั้งสองสามารถชมได้ที่ย่อหน้าแรก

ค่าความไม่ครบตอนนี้อยู่ที่ 16 คน คำนวนจากผู้มีสิทธิ์ออกเสียงประชามติทั้งหมด [82 คน](https://image.mfa.go.th/mfa/0/pFxDf7ZYQ7/vote_2026/%E0%B8%9B%E0%B8%A3%E0%B8%B0%E0%B8%81%E0%B8%B2%E0%B8%A8%E0%B8%9A%E0%B8%B1%E0%B8%8D%E0%B8%8A%E0%B8%B5%E0%B8%A3%E0%B8%B2%E0%B8%A2%E0%B8%8A%E0%B8%B7%E0%B9%88%E0%B8%AD%E0%B8%9C%E0%B8%B9%E0%B9%89%E0%B8%A1%E0%B8%B5%E0%B8%AA%E0%B8%B4%E0%B8%97%E0%B8%98%E0%B8%B4%E0%B8%AD%E0%B8%AD%E0%B8%81%E0%B9%80%E0%B8%AA%E0%B8%B5%E0%B8%A2%E0%B8%87%E0%B8%9B%E0%B8%A3%E0%B8%B0%E0%B8%8A%E0%B8%B2%E0%B8%A1%E0%B8%95%E0%B8%B4%E0%B8%99%E0%B8%AD%E0%B8%81%E0%B8%A3%E0%B8%B2%E0%B8%8A%E0%B8%AD%E0%B8%B2%E0%B8%93%E0%B8%B2%E0%B8%88%E0%B8%B1%E0%B8%81%E0%B8%A3.pdf) ลบคะแนนโพลรวมของผู้ที่ออกเสียงเห็นด้วย 53 คน และไม่เห็นด้วย 13 คน โดยประมาณแบบเผื่อว่าเป็นไปได้ว่าคนที่กดไม่สะดวกตอบไม่เห็นปุ่มสำหรับคนที่ไม่ได้ลงทะเบียนจึงใช้ปุ่มแทนกัน จึงเหลือ 16 คน (82-53-13=16) เป็นตัวเลขค่าความไม่ครบที่ใหญ่ที่สุดที่เป็นได้ โดยคาดว่าปริมาณนี้จะลดลงเมื่อสถานทูตเผยปริมาณผู้มาใช้เสียงจริง

ด้วยความที่การเลือกตั้งครั้งนี้มีการลงประชามติด้วย จึงน่าสนใจว่าคำถามสองข้อมีความเกี่ยวโยงกันหรือไม่

สังเกตได้ว่ามีความเกี่ยวโยง คนที่เลือกพรรคประชาชนส่วนใหญ่เห็นชอบกับคำถามประชามติ และคนที่เลือกพรรคภูมิใจไทยส่วนใหญ่ไม่เห็นชอบ ส่วนคนที่เลือกพรรคอื่นๆเช่น พรรคเพื่อไทย ไทยสร้างไทย และประชาธิปัตย์ ก็เห็นชอบด้วยส่วนใหญ่หรือทั้งหมด แต่ปริมาณผู้ลงเสียงน้อยเกินที่จะสรุปอะไรได้

## ความผิดพลาดและความไม่ครบถ้วนของข้อมูล

- วันนั้นเป็นอยู่ในช่วงพายุหมุนขั้วโลก (polar vortex) โคตรหนาว หนาวสติหลุด 555 พี่ๆทีมงานสถานทูตได้มีพื้นที่ๆอุ่นไว้ให้ แต่เพราะแอบอยู่ตรงนี้ เลยอาจจะทำให้ไม่เห็นบางคนที่เดินออกจากประตู ประมาณยากว่ากี่คน
- ช่วงที่คนเยอะ มีหลุดเพราะกำลังขอเสียงของอีกคนอยู่ เท่าที่เห็นและจำได้มีหนึ่งคน
- บางคนปฏิเสธที่จะทำโพล เท่าที่จำได้มีหนึ่งคน
- การยื่นมือถือให้ผู้ลงเสียงโพล ทำให้ผมไม่เห็นว่าใครกดอะไรบนมือถือ อาจจะมีคนกดเกินหนึ่งครั้ง หรือกดผิดจนเสียงไม่ถูกนับ ไม่เหมือนรอบที่แล้วที่มีเสียงกดคีย์บอร์ด คิดว่าไม่น่าพลาดเยอะเพราะแอพพลิเคชั่นใช้ง่าย แต่ก็ประมาณยากว่าพลาดเยอะเท่าไหร่ ในอนาคตแก้ได้โดยการทำให้มีเสียงตอนกด

## หมายเหตุ

ในการจัดทำโพลนี้ ได้รับความช่วยเหลือจากคนหลายกลุ่ม ซึ่งย่อมมีมุมมองหรืออคติที่ต่างกัน จึงขอใช้พื้นที่ตรงนี้ประกาศให้ชัดเจนว่า ในการทำโพลครั้งนี้ ไม่มีการแลกเปลี่ยนผลประโยชน์ และไม่มีการเอาความช่วยเหลือมาโน้มน้าวความคิดผู้เขียนคนนี้

ความเห็น มุมมอง คำวิเคราะห์ และความผิดพลาด ทุกประการเป็นความรับผิดชอบผมและผมคนเดียว

## ทิ้งท้าย

มีหลายคนตั้งคำถามว่า ทำไมต่างประเทศเมืองอื่นไม่มีแบบนี้ ผมขอตอบง่ายๆว่า เพราะไม่มีคนทำ ผมในฐานะคนธรรมดาคนหนึ่ง ทำที่เดียวก็หัวหมุนแล้ว จึงทำแค่ที่กรุงออตตาว่า

ก็ใช่ว่ามันลำบากเดินทางกับเสียเวลาทั้งวันมานั่งดักคน แต่พูดจริงคือมันไม่ได้ทำยาก ที่เราใช้เทคโนโลยีก็เพราะใช้เป็น แต่เอาจริงแค่กระดาษ ปากกา ซอง และความมุมานะ ก็ได้แล้ว 

จึงต้องเปลี่ยนคำถามเป็น ทำไมที่อื่นไม่มี เป็น แล้วทำไมเราไม่ทำ
